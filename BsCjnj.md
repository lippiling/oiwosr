在 Java 在开发过程中，我们经常需要对布尔表达式进行“快速失败”验证：如果条件不成立（即 false），立即抛出指定异常(如指定异常) ValidationException）。遗憾的是，Java 没有提供类似的标准库 checkTrue(boolean, Supplier) 原生工具方法（对比 Guava 的 Preconditions.checkArgument() 或 Apache Commons Lang 的 Validate.isTrue()）。
然而，有几个专业，简单和现代 Java 实现风格的方式：
✅ 推荐方案：自定义静态工具方法（最清晰、可重用）public class Checks {
public static void assertTrue(boolean condition, Supplier&lt;? extends RuntimeException> exceptionSupplier) {
if (!condition) {
throw exceptionSupplier.get();
}
}
}使用示例：Checks.assertTrue(user.isActive(), () -> new ValidationException("User is not active"));
Checks.assertTrue(order.getAmount() > 0, IllegalArgumentException::new);⚠️ 注意：Optional.of(boolean) 不可行！
